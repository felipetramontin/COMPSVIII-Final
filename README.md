# Workout Manager

Workout Manager is a one-page web application designed to help users track exercises, sets, reps, and weights. It streamlines workout tracking, ensuring users can easily log and manage their routines.

## Technology Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: JSON storage (local)
- **Deployment**: Local server / optional Heroku or Vercel

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/username/workout-manager.git
   ```
2. Navigate to the project folder:
   ```bash
   cd workout-manager
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   node server.js
   ```
5. Open your browser at: `http://localhost:3000`

## Features
- Add exercises with name, sets, reps, and weight
- Edit existing exercises
- Delete exercises
- Mark exercises as complete
- Persistent storage via JSON file
- Dynamic UI updates without page reload

## Usage Example
1. Fill in the exercise form and click "Add Exercise."
2. Edit or delete exercises using the respective buttons.
3. Mark exercises as complete to track progress visually.
